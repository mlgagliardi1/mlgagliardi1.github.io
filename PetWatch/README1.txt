To run this code, the user first needs to use pip to install django
    pip install -m django

Once django is installed, the user should navigate to the folder containing the manage.py file
    ..MLGAGLIARDI1.GITHUB.IO/PetWatch

Next, the user will use the django command line to run the server
    type 'py manage.py runserver' in the terminal
    The user should see the server begin running in the terminal

Finally, open up a browser and navigate to the localhost site
    Should be http://localhost:8000/PetWatch//
